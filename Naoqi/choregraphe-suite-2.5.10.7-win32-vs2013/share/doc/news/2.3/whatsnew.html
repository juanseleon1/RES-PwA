Release Notes - 2.3.0¶

Supported Operating Systems¶

Linux: Ubuntu 14.04 (Trusty Tahr)¶

Ubuntu 12.04 LTS (Precise) is not supported.

Linux Ubuntu 14.04 (Trusty Tahr) is supported.

Windows: 7 and 8.1¶

Windows XP is not supported.

Windows 7 and 8.1 are supported.

Mac OS: 10.10.2 Yosemite¶

Mac OS X versions 10.7.1 Lion, 10.8.3 Mountain Lion, and 10.9.4 Mavericks, are not supported.

Mac OS X version 10.10.2 Yosemite is supported.

For further details, see:

  • Choregraphe requirements,
  • Python SDK - Supported Operating Systems,
  • C++ SDK - Supported Operating Systems.

Interacting with Aldebaran robots¶

Calling Pepper¶

When Pepper has not yet seen you, or is not close enough to be fully listening, he may, however, understand a few words.

For further details, see: Calling Pepper.

Security¶

Aldebaran robot makes sure he is in a safe position when starting Life and adds a notification if he is not in a safe position.

Notifications¶

All system notifications which are either a warning or an error are now formatted the same way: prefixed by their severity and their ID number.

For further details, see: Event and state notifications or Event and state notifications.

New tool: Memory Backup¶

Memory Backup allows you to back up and restore personal data stored on the robot.

For further details, see:

  • Backing up / restoring robot data,
  • Back up my Aldebaran robot user data.
