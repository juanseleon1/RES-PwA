NAOqi OS 2.5¶

Miscellaneous¶

The embedded NAOqi OS has evolved; among the noticeable changes, there are:

  • NAOqi OS now uses systemd as init system;
  • Busybox now provides a number of common tools (e.g.: tar, awk, find...)
  • Linux kernel has been bumped to version 4.0;
  • The root filesystem / is now read-only;
  • NAOqi software is now located in /opt/aldebaran.
  • imaging package has been removed, please use Pillow package instead.

The whole list of packages installed in NAOqi OS is a available in: Open Source Projects.
