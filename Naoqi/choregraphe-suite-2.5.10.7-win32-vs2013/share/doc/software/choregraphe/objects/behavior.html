Behavior¶

What is a Behavior¶

Creating a Behavior is an easy way to make the robot talk, walk, dance, play a sound, etc.

A Behavior contains Boxes which are sequentially or simultaneously executed.

Inside the Project folder, the Behavior is stored in a .XAR file located in a sub-folder.

  • The description language in which you want to edit the properties.

  • The behavior’s properties.

    Note

    The trigger sentences and the loading responses are not editable in all description languages, but only in those that are supported languages too.
